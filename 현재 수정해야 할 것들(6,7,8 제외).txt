1 page
-별자리 선을 만드는 함수를 DB에서 가져온 값을 통해, 달성한 것에 맞춰서 그어주는 기능 구현 필요
-별자리를 완성했을 때 새로운 창(or 알림창)을 띄워서 쓰고 싶은 말을 적을 수 있게 해야함

2 page
-수정 기능 -> 클릭하면 목적 적는 것, 시작, 종료시간, 측정 방법이 txt필드(콤보박스 등 수정할 수 있는 것)로 변환되서 바꿀 수 있게 기능을 수정해야함
-삭제 기능 -> 클릭하면 화면에서 지워지게 하고 DB에서도 기록 삭제
-목적, 시작, 종료시간 -> 글자로 변환
-타이머 기능 첨가

3 page
DB를 통해서 사용자가 만든 별자리의 사진을 버튼으로 만들 수 있게 기능 수정

4 page
날짜, 설명 => 구현한 날짜를 적게 해야함, 별자리가 어떻게 생겼는지 설명을 적게 해줘야 함
쓰고 싶은 말 => 별자리를 완성 시, 적었던 말을 넣어줘야 함

5 page
-각 버튼의 년, 월, 일의 인덱스를 받아서 DB로 보내준 후, DB에서 저장되어 있는 목표, 시간, 측정방법을 아래에 띄워줄 수 있어야 한다.
-아래 패널의 북쪽에는 어떤 날을 표시해주고 있는지를, 아래 패널 남쪽에는 완료 버튼과 수정을 누르면 수정을 할 수 있게 구성을 다시 해줘야 할 것 같다. 수정을 누르면 현재 나와있는 것처럼 txt필드가 나오게 하고 원래는 label형태로 구성.(완료 버튼을 누르면 label 형태로 돌아가게 한다.)

현재 문제점 
- 2,5 panel에서 콤보박스를 누르면 콤보박스가 지나간 부분이 사라지는 현상 발생.(2 panel같은 경우는 처음 화면에서는 그러지 않으니 다른 페이지를 넘기고 다시 돌아와서 콤보박스를 선택하면 그런 현상이 발생) => 어떻게 해결을 해야할지 알 수 가 없어서 다 같이 찾아줬으면 한다.
- 만약 이 프로그램을 아래로 내리고 다시 올리면 화면이 모두 지워진다.
